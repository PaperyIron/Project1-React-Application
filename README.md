# DND Race Information Getter

This site uses a dungeons and dragons(DND) api to fetch essential information about character races.  This site only provides quick and basic details.  To get even more data for advanced traits and details, use the official DND website.

## To Run

1. Clone repository to your local machine.
2. Open a terminal in the current folder.
3. Run 'cd dndApplication/'.
4. Run 'npm install' to install required dependencies.
5. Run 'npm run dev' to start local vite server.
6. Either click the vite link provided or visit 'http://localhost:5173/' in your browser to view the webpage.
7. After you are finished, type 'ctrl-c' to stop the server.

## How to Use
1. Click the search bar again to display all options, or simply begin typing to select your DND race.
2. Click search or press enter to search.
3. Read about your character's race!
4. Type in the search to view a different race.
